# Niall's Dotfiles

This repo contains the configuration files I use across my systems.

## Setup Scripts

There are scripts to automate the setup process on a new machine.
At this time only Mac and Windows is supported, but the Mac script might work on Linux.

### Mac

Clone the reposity and cd into it, then run:

```
./setup-mac.sh
```

This script will:

1. Install Homebrew.
2. Install the apps I use.
